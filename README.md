00後整頓職場_企業永續發展的核心動力
「迷惘，是因為選擇太多。」這是一個專為青年設計的職涯探索平台，旨在透過數據與透明資訊，縮短產學落差，並帶領使用者看見企業在 SDG 8（合適的工作與經濟成長） 上的實踐。
1.專案特點:
互動式職涯測驗：內建 30 道情境題目，結合 MBTI 特質分析，從 20 種以上跨領域職缺中推薦最適合的路徑。
深度職業百科：不僅提供職稱，更深入解析技能地圖 (Roadmap)、月薪範圍與關鍵特質，並內建「技能術語百科」。
企業永續透視：透過互動式圖卡，展示台積電、Patagonia 等指標企業如何落實 ESG 與員工福祉，將 SDG 8 從口號變為具體數據。
2.極致 UI/UX 體驗：
毛玻璃設計：打造輕盈、通透的現代視覺感。
深色模式：一鍵切換，適應不同使用場景。
響應式設計：完美適配手機與電腦瀏覽。
3.核心功能說明:
職涯測驗系統:
使用者完成測驗後，系統會計算 60 個以上的變數得分，自動排序並推薦前三名職缺。同時提供「回答回顧」，幫助使用者進行自我察覺與反思。
職業篩選百科:
支援雙重篩選機制：
職缺類別：從科技硬體、軟體開發到藝術媒體、永續影響力。
MBTI 特質：直接過濾出符合特定性格特質的職位。
學習地圖:
每個職位皆提供階梯式的學習建議。點擊關鍵技能（如：React、碳盤查）會彈出詳細的術語解釋，降低新手的進入門檻。
